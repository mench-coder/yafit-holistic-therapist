# ספריית נכסים ויזואליים — פרויקט יפית

> נוצר 2026-06-22 · אוסף אייקונים, סמלים, תמונות ואיורים לשימוש עתידי בנכסי הפרויקט (כרטיסים, פלייר, אתר, PDF).

## מיקום
כל הנכסים הגולמיים שהלקוח אוסף נשמרים תחת **`assets/library/`**.

## מוסכמות
- שם קובץ תיאורי באנגלית עם מקפים, לפי סוג: `illustration-*`, `icon-*`, `symbol-*`, `photo-*`.
- כל פריט מתועד בטבלה למטה: מה זה, מאיפה, ומגבלות שימוש.
- **לפני שימוש בפועל** — לבדוק את עמודת "סטטוס/רישיון". פריט עם סימן מים (watermark) הוא תצוגה מקדימה בלבד; צריך גרסה נקייה/מורשית לפני הטמעה בנכס מפורסם.

## הערות להמשך
- כשהלקוח יבקש "לגעת" בנכס — להתחיל מהקובץ הקיים כאן, לא מאפס.
- אם פריט דורש ניקוי רקע / וקטור / התאמת צבע למותג (מרווה/חימר/שמנת) — לבצע ולשמור גרסה נגזרת לצד המקור (למשל `*-sage.svg`).
